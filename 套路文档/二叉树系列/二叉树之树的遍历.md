# 套路代码
## 递归套路代码

```java
class Solution {
   public List<Integer> preorderTraversal(TreeNode root) {
      // 创建接收结果的集合
      List<Integer> list = new ArrayList<>();
      // 逻辑处理
      preorder(list, node);
      // 返回结果
      return list;
    }
  
  private void preorder(List<Integer> list, TreeNode node) {
    // 临界值处理
    if (node == null) {
      return;
    }
    // 由于前序遍历是在获取当前节点值之后才进行的左子节点的遍历，因此应该在此处进行填充list操作
    // 参考
    // 递归左子节点
    preorder(list, node.left);
    // 由于中序遍历是在左子节点遍历后右子节点遍历前获取当前节点值，因此应该在此处进行填充list操作
    // 递归右子节点
    preorder(list, node.right);
    // 由于后序遍历是在左右子节点都遍历后获取当前节点的值，因此应该在此处进行填充list操作
  }
}
```
